# เงินฉัน — แอปมอนิเตอร์พอร์ตการเงินส่วนตัว

แดชบอร์ดการเงินแบบมินิมอล ดูภาพรวมง่ายๆ เน้นกราฟและการทำนายแนวโน้ม
ทำเป็นเว็บไฟล์เดียว (PWA) ต่อยอดเป็นแอป iPhone ได้

## เปิดใช้งาน
- **บนคอม:** ดับเบิลคลิก `index.html` เปิดในเบราว์เซอร์ได้เลย ข้อมูลเก็บในเครื่อง (localStorage)
- ครั้งแรกกดปุ่ม **"โหลดข้อมูลตัวอย่าง"** เพื่อดูหน้าตากราฟ แล้วลบ/แก้ไขเป็นของตัวเองได้

## เมนู
1. **ภาพรวม** — การ์ดสรุป (มูลค่าสุทธิ / หนี้คงเหลือ / เงินเก็บ / คงเหลือเดือนนี้) **กดดูที่มาการคำนวณได้**,
   **ปฏิทินรอบเดือน 15→15** (ลูกศร ▲บวก/▼ติดลบ แต่ละรอบ · กดดูรายละเอียด),
   กราฟรายรับ-รายจ่ายรายปี (แกน Y +/− · ภาระเป็นแท่งลง · **เปิด/ปิดเส้นได้**),
   **กราฟแนวโน้มทรัพย์สินสะสม** ทำนาย 24 เดือน (ติ๊กรวม/ไม่รวมเงินเก็บได้),
   เส้นแนวโน้มหนี้, โดนัทสัดส่วนสินทรัพย์,
   **แผงรายละเอียดรายเดือน** เลือกเดือนดูว่าต้องจ่ายอะไรบ้าง (บัตรวันตัดไม่ตรงกันก็เห็นครบ),
   ปุ่มบันทึกยอดใช้จ่ายเดือนนี้
2. **บัตร/สินเชื่อ** — เพิ่มบัตร (บัตรเครดิต / บัตรกดเงินสด / สินเชื่อ · วงเงิน · วันตัดบัตร · วันครบกำหนด · ค่า X รายเดือน) + รายการผ่อน (ยอด/เดือน × จำนวนงวด) → บอกวันผ่อนหมด
3. **รายรับ-จ่าย** — 2 ส่วน: **รายได้** (เงินเดือน/โบนัส) และ **รายจ่าย/ภาระที่ไม่ใช่บัตร** (ค่าไฟ ค่าน้ำ ค่าเช่าบ้าน ค่าโทรศัพท์ ฯลฯ) · แต่ละรายการเลือกได้ว่า **ประจำทุกเดือน** หรือ **เฉพาะกิจเดือนเดียว** (ค่าซ่อม/ค่าเทอม) · รวมเข้ากราฟ/ปฏิทิน/รายละเอียดอัตโนมัติ
4. **อสังหาฯ** — ค่างวด vs ค่าเช่า − ส่วนกลาง = กระแสเงินสด + ทำนายรายได้/ปี
5. **หุ้น** — เงินลงทุน + กรอกกำไร/ขาดทุนรายเดือน

## แนวคิดค่า X
บัตรแต่ละใบล็อก "ค่า X" = ยอดใช้จ่ายรายเดือนที่ไม่ใช่รายการผ่อน (กาแฟ/น้ำมัน/อาหาร)
คาดเดายาก → สิ้นเดือนมากรอกยอดจริงผ่านปุ่ม **"บันทึกยอดใช้จ่ายเดือนนี้"** (กรอกย้อนหลังได้ กราฟอัปเดตทันที)

## สำรองข้อมูล
กดไอคอน ⚙️ มุมขวาบน → ส่งออก/นำเข้าไฟล์ `.json` (ควรสำรองเป็นระยะ)

## ต่อยอดเป็นแอป iPhone
ข้อมูลอยู่ในเครื่อง ถ้าจะใช้บน iPhone แบบ "Add to Home Screen" ต้องเอาโฟลเดอร์นี้ไปวางบนเว็บ (เช่น GitHub Pages / Netlify — ฟรี) แล้วเปิดผ่าน Safari → แชร์ → เพิ่มลงในหน้าจอโฮม จะได้ไอคอนและเปิดเต็มจอเหมือนแอปจริง (มี `manifest.json` + `icon.svg` เตรียมไว้แล้ว)

> หมายเหตุ: ตัวเลขทั้งหมดเป็นแบบง่าย ไม่คิดดอกเบี้ยให้อัตโนมัติ — ผู้ใช้กรอกยอดรายเดือน (รวมดอกเบี้ยที่คิดมาแล้ว) เอง
